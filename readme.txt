Plugins Needed: None

== Description ==

Built for the purpose of completing the take-home assessment for the Black Duck Digital Developer position.

Download the whole Github repo as a zip file, this includes all theme files needed
In WordPress, navigate to Appearance > Themes and click ‘Add Theme’ button at the top
Click ‘Upload Theme’ button
Click ‘Choose File’ and select to upload the zipped theme folder, then click ‘Install’
Once installation is complete, click to activate the theme
Navigate to Pages and create a new page and give it a name
In the sidebar, click to change the template and select the ‘eBook Redesign’ template
The template & pattern is totally set up so you can just publish and view the page without having to change anything

Demo can be viewed at https://playground.wordpress.net/?blueprint-url=https://github.com/mck-debandi/black-duck-demo/blob/main/blueprint.json
